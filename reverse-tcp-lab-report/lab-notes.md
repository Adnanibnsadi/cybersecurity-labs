# Reverse TCP Lab

## Overview

This lab was conducted in a controlled virtual environment to study remote TCP-based communication between systems and understand the associated security implications.

## Objectives

- Understand how a remote connection can be established between two virtual machine in a controlled lab.
- Explore network communication between virtual machines.
- Observe how remote connections can be established.
- Understand the risks of executing untrusted files.
- Learn about monitoring and detecting suspicious network activity.

## Lab Environment

| Component | Description |
|------------|------------|
| Host System | Windows 11 |
| Virtualization Platform | VMware Workstation |
